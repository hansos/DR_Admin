# POST SetAllCountriesActive

Manages countries and their information

## Endpoint

```
POST /api/v1/countries/set-all-active/{isActive}
```

## Authorization

Requires authentication. Policy: **Geographical.Write**.

## Parameters

| Name | Source | Type |
|------|--------|------|
| `isActive` | Route | `bool` |
