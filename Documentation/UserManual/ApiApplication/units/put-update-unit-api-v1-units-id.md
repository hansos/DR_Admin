# PUT UpdateUnit

Update an existing unit

## Endpoint

```
PUT /api/v1/units/{id}
```

## Authorization

Requires authentication. Policy: **Authenticated**.

## Parameters

| Name | Source | Type |
|------|--------|------|
| `id` | Route | `int` |
| `updateDto` | Body | `UpdateUnitDto` |
