# GET GetRegistrarById

Retrieves all domain registrars in the system

## Endpoint

```
GET /api/v1/registrars/{id}
```

## Authorization

Requires authentication. Policy: **Registrar.Read**.

## Parameters

| Name | Source | Type |
|------|--------|------|
| `id` | Route | `int` |
