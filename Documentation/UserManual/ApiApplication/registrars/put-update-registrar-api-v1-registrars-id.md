# PUT UpdateRegistrar

Update an existing registrar

## Endpoint

```
PUT /api/v1/registrars/{id}
```

## Authorization

Requires authentication. Policy: **Authenticated**.

## Parameters

| Name | Source | Type |
|------|--------|------|
| `id` | Route | `int` |
| `updateDto` | Body | `UpdateRegistrarDto` |
