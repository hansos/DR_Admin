# GET GetSalesSummary

Retrieves dashboard sales summary with offers, orders and open invoices

## Endpoint

```
GET /api/v1/system/sales-summary
```

## Authorization

Requires authentication. Policy: **Authenticated**.

## Responses

| Code | Description | Body |
|------|-------------|------|
| 200 | OK | - |
| 401 | Unauthorized | - |
| 500 | Internal Server Error | - |
