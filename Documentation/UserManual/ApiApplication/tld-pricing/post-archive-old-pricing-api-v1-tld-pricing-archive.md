# POST ArchiveOldPricing

Gets all TLDs with negative margins (cost > price)

## Endpoint

```
POST /api/v1/tld-pricing/archive
```

## Authorization

Requires authentication. Policy: **Pricing.Admin**.

## Responses

| Code | Description | Body |
|------|-------------|------|
| 200 | OK | `object` |
