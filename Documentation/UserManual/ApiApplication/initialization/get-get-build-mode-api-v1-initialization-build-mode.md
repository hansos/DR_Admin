# GET GetBuildMode

Gets whether the API runs in Debug mode for initialization-time debug tooling.

## Endpoint

```
GET /api/v1/initialization/build-mode
```

## Authorization

This endpoint does not require authentication.

## Responses

| Code | Description | Body |
|------|-------------|------|
| 200 | OK | - |
