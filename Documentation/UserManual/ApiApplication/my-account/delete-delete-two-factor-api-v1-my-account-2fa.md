# DELETE DeleteTwoFactor

Deletes all two-factor authentication settings for the authenticated user.

## Endpoint

```
DELETE /api/v1/my-account/2fa
```

## Responses

| Code | Description | Body |
|------|-------------|------|
| 200 | OK | - |
| 400 | Bad Request | - |
| 401 | Unauthorized | - |
| 500 | Internal Server Error | - |
