# PUT UpdateTld

Update an existing TLD

## Endpoint

```
PUT /api/v1/tlds/{id}
```

## Authorization

Requires authentication. Policy: **Authenticated**.

| Name | Source | Type |
|------|--------|------|
| `id` | Route | `int` |
