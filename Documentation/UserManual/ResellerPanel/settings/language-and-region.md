# Language and Region

Configure language, locale, date format, and regional formatting preferences.

## How to Access

Navigate to **Settings > Language & Region** from the side menu.

## Configuration Fields

| Setting | Description |
|---------|-------------|
| Language | Interface language for the panel. |
| Date Format | How dates are displayed (e.g., DD/MM/YYYY, MM/DD/YYYY, YYYY-MM-DD). |
| Time Format | 12-hour or 24-hour clock. |
| Timezone | System timezone for displaying dates and scheduling. |
| Number Format | Decimal and thousands separators (e.g., 1,000.00 vs 1.000,00). |
| Currency Position | Symbol before or after the amount. |

## Saving Changes

Click **Save** to apply. Formatting changes affect the entire panel interface.
