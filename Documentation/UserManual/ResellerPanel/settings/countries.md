# Countries

Manage the list of countries available throughout the system.

## How to Access

Navigate to **Settings > Countries** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Country Name | Full country name. |
| ISO Code | Two-letter ISO 3166-1 alpha-2 code. |
| Calling Code | International dialing code. |
| Enabled | Whether this country is available for selection in forms. |
| Actions | Edit, Enable/Disable. |
