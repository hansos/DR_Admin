# Deadlines

Set default deadlines for renewals, transfers, and other time-sensitive operations.

## How to Access

Navigate to **Settings > Deadlines** from the side menu.

## Configuration Fields

| Setting | Description |
|---------|-------------|
| Renewal Reminder (days) | How many days before expiry to send the first renewal reminder. |
| Renewal Deadline (days) | Last day before expiry that a renewal can be processed. |
| Invoice Advance (days) | How many days before a service period to generate invoices. |
| Payment Due (days) | Default number of days after invoice date for payment to be due. |
| Transfer Response (days) | Deadline to approve or reject an outgoing transfer request. |
| Quote Validity (days) | How many days a quote remains valid before expiring. |

## Saving Changes

Click **Save** to apply. These values are used as defaults for new operations and can be overridden per-item where applicable.
