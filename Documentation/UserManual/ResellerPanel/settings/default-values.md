# Default Values

Set system-wide default values for forms, operations, and new records.

## How to Access

Navigate to **Settings > Default Values** from the side menu.

## Configuration Fields

| Setting | Description |
|---------|-------------|
| Default Currency | Currency pre-selected in new invoices and orders. |
| Default Billing Cycle | Billing period pre-selected for new domains. |
| Default Nameservers | Nameserver set assigned to new registrations. |
| Default DNS Template | DNS template applied to new domains. |
| Default Payment Instrument | Payment method pre-selected for new customers. |
| Default Tax Category | Tax category assigned to new products. |
| Default Registration Period | Number of years pre-selected for new registrations. |

## Saving Changes

Click **Save** to apply. Defaults are used for new records only and do not retroactively change existing data.
