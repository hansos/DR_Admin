# Domain Reports

Generate reports on domain portfolio metrics, registration trends, and expiration forecasts.

## How to Access

Navigate to **Reports & Export > Domain Reports** from the side menu.

## Available Reports

| Report | Description |
|--------|-------------|
| Portfolio Summary | Total domains by status (active, expired, pending). |
| Registrations Over Time | New registrations per period (monthly, quarterly, yearly). |
| Expiration Forecast | Domains expiring in upcoming periods. |
| TLD Distribution | Domain count per top-level domain. |
| Customer Distribution | Domain count per customer. |

## Generating a Report

1. Select the report type.
2. Configure filters (date range, TLD, customer, status).
3. Click **Generate**.
4. View the results as a table or chart.
5. Optionally export via the [Export](export.md) page.
