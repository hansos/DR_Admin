# Export

Export data in various formats (CSV, Excel, PDF) for external use.

## How to Access

Navigate to **Reports & Export > Export** from the side menu.

## Available Export Types

| Data | Formats |
|------|---------|
| Domain List | CSV, Excel |
| Customer List | CSV, Excel |
| Invoices | CSV, Excel, PDF |
| Payments | CSV, Excel |
| Reports | CSV, Excel, PDF |

## Step-by-Step

1. Select the data type to export.
2. Configure filters (date range, status, etc.).
3. Choose the output format.
4. Click **Export**.
5. Download the generated file.

## Scheduled Exports

Set up recurring exports that are generated automatically and emailed or saved to a configured location.
