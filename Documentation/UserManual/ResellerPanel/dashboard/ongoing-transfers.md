# Ongoing Transfers

Monitor domain transfers currently in progress.

## How to Access

Navigate to **Dashboard > Ongoing Transfers** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Domain Name | The domain being transferred. |
| Direction | Incoming or Outgoing. |
| Status | Current transfer status (Pending, Awaiting Approval, In Progress, etc.). |
| Initiated | Date the transfer was started. |
| Customer | Associated customer. |
| Actions | View Details, Cancel, Retry. |

## Transfer Statuses

| Status | Meaning |
|--------|---------|
| Pending | Transfer request submitted, awaiting registry processing. |
| Awaiting Approval | Waiting for the losing registrar or registrant to approve. |
| In Progress | Registry is processing the transfer. |
| Failed | Transfer was rejected or timed out. See [Status & Errors](../transfers/status-and-errors.md). |
