# Expiring Domains

View domains approaching their expiration date and take renewal action.

## How to Access

Navigate to **Dashboard > Expiring Domains** from the side menu.

## Page Layout

A filtered table showing domains sorted by expiration date (soonest first).

| Column | Description |
|--------|-------------|
| Domain Name | The fully qualified domain name. |
| Customer | The customer who owns the domain. |
| Expiry Date | Date the domain registration expires. |
| Days Left | Number of days until expiration. |
| Auto-Renew | Whether automatic renewal is enabled. |
| Actions | Renew, Remind Customer, or View Details. |
