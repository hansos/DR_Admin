# Unpaid Invoices

List of outstanding invoices requiring payment or follow-up.

## How to Access

Navigate to **Dashboard > Unpaid Invoices** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Invoice # | Unique invoice identifier. |
| Customer | Customer the invoice belongs to. |
| Issue Date | Date the invoice was created. |
| Due Date | Payment deadline. |
| Amount | Total amount due. |
| Status | Unpaid, Overdue, or Partially Paid. |
| Actions | View, Send Reminder, Record Payment. |
