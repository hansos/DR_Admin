# New Quote

Create a new domain registration, transfer, or renewal quote for a customer.

## How to Access

Navigate to **Dashboard > New Quote** from the side menu.

## Quote Status

| Status   | Meaning                                            |
| -------- | -------------------------------------------------- |
| Draft    | Quote created but not yet sent to the customer.    |
| Sent     | Quote emailed or shared with the customer.         |
| Accepted | Customer accepted; ready to convert to an order.   |
| Expired  | Quote passed its validity date without acceptance. |

## What Happens Next

- An accepted quote can be converted into an **Order** from the [Orders](../billing/orders.md) page.
- The customer receives an email notification (if configured in [Templates](../alerts/templates.md)).
