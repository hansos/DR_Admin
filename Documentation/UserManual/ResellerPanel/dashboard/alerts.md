# Alerts

View important notifications and system alerts requiring attention.

## How to Access

Navigate to **Dashboard > Alerts** from the side menu.

## Page Layout

A list of alerts sorted by severity and date, showing:

| Column | Description |
|--------|-------------|
| Severity | Critical, Warning, or Info. |
| Message | Short description of the alert. |
| Source | The system area that generated the alert (e.g., Billing, DNS, Transfers). |
| Date | When the alert was triggered. |
| Actions | Acknowledge, View Details, Dismiss. |
