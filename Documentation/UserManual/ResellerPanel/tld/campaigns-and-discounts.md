# Campaigns and Discounts

Manage promotional campaigns, seasonal offers, and TLD-specific discounts.

## How to Access

Navigate to **TLD & Registry > Campaigns & Discounts** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Campaign Name | Name of the promotional campaign. |
| TLDs | TLDs included in the campaign. |
| Discount | Percentage or fixed-amount discount. |
| Valid From | Campaign start date. |
| Valid To | Campaign end date. |
| Status | Active, Scheduled, or Expired. |
| Actions | Edit, Disable, Delete. |

## Creating a Campaign

1. Click **Add Campaign**.
2. Enter a campaign name and description.
3. Select one or more TLDs.
4. Set the discount type and value.
5. Define the validity period.
6. Click **Save**.
