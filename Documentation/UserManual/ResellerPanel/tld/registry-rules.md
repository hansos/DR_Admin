# Registry Rules

View registration rules, eligibility requirements, and policies per TLD registry.

## How to Access

Navigate to **TLD & Registry > Registry Rules** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| TLD | The top-level domain. |
| Min Period | Minimum registration period (years). |
| Max Period | Maximum registration period (years). |
| Eligibility | Any restrictions on who can register (e.g., local presence). |
| DNSSEC Required | Whether DNSSEC is mandatory. |
| Transfer Lock | Lock period after registration or transfer. |
| Special Rules | Additional registry-specific requirements. |

## Using This Information

- Consult these rules before initiating registrations or transfers to ensure compliance.
- Rules are maintained by the system administrator based on current registry policies.
