# Roles and Permissions

Define roles and assign granular permissions to control user access across the panel.

## How to Access

Navigate to **Users & Roles > Roles & Permissions** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Role Name | Name of the role (e.g., Admin, Manager, Support). |
| Users | Number of users assigned to this role. |
| Permissions | Summary of granted permissions. |
| Actions | Edit, Duplicate, Delete. |

## Creating a Role

1. Click **Add Role**.
2. Enter a role name and description.
3. Select permissions from the permission matrix:

| Permission Area | Available Permissions |
|-----------------|----------------------|
| Domains | View, Create, Edit, Delete, Transfer. |
| Customers | View, Create, Edit, Delete. |
| Billing | View Invoices, Create Invoices, Manage Payments, Manage Pricing. |
| DNS | View, Create, Edit, Delete zones and records. |
| Settings | View, Edit system settings. |
| Users | View, Create, Edit, Delete users and roles. |

4. Click **Save**.

## Built-in Roles

| Role | Description |
|------|-------------|
| Administrator | Full access to all features. |
| Manager | All features except user/role management and system settings. |
| Support | Read-only access with ability to manage domains and DNS. |
