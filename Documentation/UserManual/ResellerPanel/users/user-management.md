# User Management

Create, edit, and manage user accounts for the reseller panel.

## How to Access

Navigate to **Users & Roles > User Management** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Username | The user's login name. |
| Full Name | Display name. |
| Email | User's email address. |
| Role | Assigned role (from [Roles & Permissions](roles-and-permissions.md)). |
| Status | Active, Suspended, or Locked. |
| Last Login | Date and time of the most recent login. |
| Actions | Edit, Suspend, Reset Password, Delete. |

## Creating a User

1. Click **Add User**.
2. Enter username, full name, and email.
3. Assign a role.
4. Set a temporary password or send an invitation email.
5. Click **Save**.
