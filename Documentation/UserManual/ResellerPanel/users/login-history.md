# Login History

Review login activity and session history for all users.

## How to Access

Navigate to **Users & Roles > Login History** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Date/Time | When the login occurred. |
| Username | User who logged in. |
| IP Address | IP address of the login. |
| Location | Approximate location based on IP. |
| Status | Success, Failed, or Locked Out. |
| User Agent | Browser/device information. |

- Repeated failed login attempts trigger account lockout (configurable in [Security](../settings/security.md)).
- Suspicious activity (e.g., logins from unusual locations) may generate alerts in [Alerts](../dashboard/alerts.md).
