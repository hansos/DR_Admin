# Bulk Operations

Perform actions on multiple domains at once, such as bulk renewals, DNS changes, or registrant updates.

## How to Access

Navigate to **Domains > Bulk Operations** from the side menu, or select multiple domains from the [Domain List](domain-list.md) and click **Bulk Actions**.

## Available Bulk Operations

| Operation | Description |
|-----------|-------------|
| Bulk Renew | Renew multiple domains for a selected period. |
| Bulk DNS Update | Apply a [DNS Template](../dns/dns-templates.md) to multiple domains. |
| Bulk Nameserver Change | Change nameservers for multiple domains at once. |
| Bulk Registrant Change | Update the registrant contact on multiple domains. |
| Bulk Lock/Unlock | Enable or disable transfer lock. |
| Bulk Auto-Renew | Toggle auto-renewal on or off. |
