# Registrants

Manage domain registrant (owner) contact information and handle registrant changes.

## How to Access

Navigate to **Domains > Registrants** from the side menu.

## Page Layout

A list of registrant contact profiles used across your domain portfolio.

| Column | Description |
|--------|-------------|
| Name | Registrant name (person or organization). |
| Organization | Company or organization name. |
| Email | Contact email address. |
| Country | Country of the registrant. |
| Domains | Number of domains using this registrant profile. |
| Actions | View, Edit, Delete. |

## Creating a Registrant

1. Click **Add Registrant**.
2. Fill in the required contact fields (name, address, email, phone).
3. Click **Save**.

## Changing a Domain's Registrant

1. Open the domain in [Domain Details](domain-details.md).
2. In the **Contacts** section, click **Change Registrant**.
3. Select an existing registrant profile or create a new one.
4. Confirm the change. Some registries require email verification.

## Important Notes

- Registrant changes may trigger a 60-day transfer lock depending on the registry (see [Registry Rules](../tld/registry-rules.md)).
- Some TLDs require identity verification for registrant changes.
