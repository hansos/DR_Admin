# Domain List

Browse, search, and filter all domains in your portfolio.

## How to Access

Navigate to **Domains > Domain List** from the side menu.

## Page Layout

A paginated table displaying all domains under your reseller account.

| Column | Description |
|--------|-------------|
| Domain Name | Fully qualified domain name. |
| TLD | Top-level domain extension. |
| Customer | Customer who owns the domain. |
| Status | Active, Expired, Pending, Suspended, etc. |
| Expiry Date | When the domain registration expires. |
| Auto-Renew | Whether auto-renewal is enabled. |
| Actions | View, Edit, Renew, Transfer. |
