# Change History

Review a log of all DNS and nameserver changes with timestamps and user attribution.

## How to Access

Navigate to **DNS & Nameservers > Change History** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Date/Time | When the change was made. |
| Domain | The domain affected. |
| Change Type | Record Added, Record Modified, Record Deleted, Nameserver Changed. |
| Details | Specific fields that were changed (old value to new value). |
| User | The user who made the change. |
