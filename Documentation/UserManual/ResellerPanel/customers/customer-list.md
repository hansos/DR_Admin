# Customer List

Browse, search, and manage all customer accounts in your reseller portfolio.

## How to Access

Navigate to **Customers > Customer List** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Customer Name | Name of the customer (person or company). |
| Email | Primary contact email. |
| Domains | Number of domains owned by this customer. |
| Balance | Current account balance or outstanding amount. |
| Status | Active, Suspended, or Closed. |
| Actions | View, Edit, Suspend, Delete. |

## Creating a Customer

1. Click **Add Customer**.
2. Fill in customer details (name, email, address, phone).
3. Optionally assign a billing currency and default payment method.
4. Click **Save**.
