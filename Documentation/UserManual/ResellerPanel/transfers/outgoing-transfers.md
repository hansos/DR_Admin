# Outgoing Transfers

Manage domains being transferred out of your reseller account to other registrars.

## How to Access

Navigate to **Transfers > Outgoing Transfers** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Domain Name | The domain being transferred out. |
| Customer | The domain's current customer. |
| Status | Pending Approval, Approved, Rejected, Completed. |
| Requested | Date the transfer-out was requested. |
| Actions | Approve, Reject, View. |

## Important Notes

- The domain must be unlocked before an outgoing transfer can proceed. Disable the transfer lock in [Domain Details](../domains/domain-details.md).
- An authorization code must be provided to the gaining registrar. Generate one in [Auth Codes](auth-codes.md).
