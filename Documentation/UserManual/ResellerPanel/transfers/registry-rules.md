# Registry Rules

View transfer policies, lock periods, and requirements per domain registry.

## How to Access

Navigate to **Transfers > Registry Rules** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| TLD | Top-level domain (e.g., .com, .net, .eu). |
| Auth Code Required | Whether an EPP auth code is needed. |
| Transfer Lock Period | Mandatory lock period after registration or transfer. |
| Approval Method | Auto-approve, Email approval, or Manual. |
| Min Registration Period | Minimum period the domain must be registered before transfer. |
| Special Requirements | Any additional requirements (e.g., local presence). |

## Notes

- Registry rules are maintained by the system administrator and reflect current registry policies.
- Rules may change when registries update their policies. Check periodically for updates.
