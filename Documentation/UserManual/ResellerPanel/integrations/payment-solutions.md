# Payment Solutions

Manage integrations with payment service providers for processing transactions.

## How to Access

Navigate to **Integrations > Payment Solutions** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Provider | Payment provider name (e.g., Stripe, Nets, PayPal). |
| Integration Type | API, Redirect, or Hosted Page. |
| Status | Active, Disabled, Error. |
| Actions | Edit, Test, Disable, Delete. |

## Configuring a Provider

1. Click **Add Provider**.
2. Select the payment provider.
3. Enter API credentials and configuration.
4. Map to one or more [Payment Gateways](../billing/payment-gateways.md).
5. Click **Save & Test**.
