# Accounting

Configure integrations with external accounting and ERP systems.

## How to Access

Navigate to **Integrations > Accounting** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| System Name | Accounting system name (e.g., Tripletex, Xero, QuickBooks). |
| Sync Direction | One-way (export) or Two-way (sync). |
| Last Sync | Date and time of the last synchronization. |
| Status | Active, Disabled, Error. |
| Actions | Edit, Sync Now, View Log, Disable, Delete. |

## Configuring an Integration

1. Click **Add Integration**.
2. Select the accounting system.
3. Enter API credentials or authorize via OAuth.
4. Configure data mapping (accounts, tax codes, customers).
5. Set sync frequency (manual, daily, real-time).
6. Click **Save**.
