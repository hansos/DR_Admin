# Email and SMS

Set up email and SMS delivery providers for sending notifications and alerts.

## How to Access

Navigate to **Integrations > Email & SMS** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Provider | Provider name (e.g., SendGrid, Mailgun, Twilio). |
| Type | Email, SMS, or Both. |
| Status | Active, Disabled, Error. |
| Default | Whether this is the default delivery provider. |
| Actions | Edit, Test, Disable, Delete. |

## Configuring a Provider

1. Click **Add Provider**.
2. Select the provider type (Email or SMS).
3. Enter API credentials.
4. Configure sender address/number.
5. Click **Save & Test** to send a test message.
