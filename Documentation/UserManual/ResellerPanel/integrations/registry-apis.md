# Registry APIs

Configure connections to domain registry API providers (EPP/API credentials and endpoints).

## How to Access

Navigate to **Integrations > Registry APIs** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Registry Name | Name of the registry provider. |
| Protocol | EPP, REST API, or other. |
| Endpoint | API endpoint URL. |
| Status | Connected, Disconnected, Error. |
| TLDs | TLDs managed through this connection. |
| Actions | Edit, Test Connection, Disable, Delete. |

## Adding a Registry Connection

1. Click **Add Registry**.
2. Enter the registry name and protocol.
3. Provide the endpoint URL and credentials (username, password, or API key).
4. Map the TLDs managed by this registry.
5. Click **Save & Test** to verify connectivity.
