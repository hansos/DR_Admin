# Debug

Access diagnostic tools and debug information (visible to administrators only).

## How to Access

Navigate to **Help & Support > Debug** from the side menu. This page is only visible to users with administrator privileges.

## Available Tools

| Tool | Description |
|------|-------------|
| System Info | View application version, runtime, and environment details. |
| Configuration Dump | Display current system configuration (sensitive values masked). |
| API Health Check | Test connectivity to all configured APIs (registries, payment, email). |
| Log Viewer | Browse recent application logs. |
| Cache Management | View and clear application caches. |

## Important Notes

- This page contains sensitive diagnostic information. Access is restricted to administrators.
- Use the **API Health Check** to quickly verify all external integrations are functioning.
