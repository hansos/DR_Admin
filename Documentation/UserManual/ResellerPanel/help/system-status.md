# System Status

View current system health, uptime, and service status.

## How to Access

Navigate to **Help & Support > System Status** from the side menu.

## Page Layout

### Service Status

| Service | Description |
|---------|-------------|
| Reseller Panel | Web application status. |
| API | Backend API availability. |
| DNS Services | DNS server status. |
| Payment Processing | Payment gateway connectivity. |
| Email Delivery | Email sending service status. |

### Status Indicators

| Indicator | Meaning |
|-----------|---------|
| Operational | Service is running normally. |
| Degraded | Service is experiencing reduced performance. |
| Partial Outage | Some functionality is unavailable. |
| Major Outage | Service is down. |
| Maintenance | Planned maintenance in progress. |

### Uptime History

A chart showing uptime percentage over the last 30/60/90 days.

### Incident History

A log of recent incidents with timestamps, affected services, and resolution notes.
