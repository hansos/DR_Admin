# Support Tickets

Create and manage support requests with the DR Admin support team.

## How to Access

Navigate to **Help & Support > Support Tickets** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Ticket # | Unique ticket identifier. |
| Subject | Brief description of the issue. |
| Priority | Low, Normal, High, Urgent. |
| Status | Open, In Progress, Awaiting Reply, Resolved, Closed. |
| Created | Date the ticket was opened. |
| Last Updated | Date of the most recent activity. |
| Actions | View, Reply, Close. |

## Creating a Ticket

1. Click **New Ticket**.
2. Enter a subject and detailed description.
3. Select the priority level.
4. Optionally attach files (screenshots, logs).
5. Click **Submit**.
