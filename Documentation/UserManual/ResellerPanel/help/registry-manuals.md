# Registry Manuals

Access registry-specific documentation, policies, and technical guidelines.

## How to Access

Navigate to **Help & Support > Registry Manuals** from the side menu.

## Page Layout

A collection of documentation organized by registry and TLD:

| Column | Description |
|--------|-------------|
| Registry | Registry operator name. |
| TLDs | TLDs managed by this registry. |
| Documents | Available manuals and policy documents. |
| Actions | View, Download. |

## Typical Documents

- Registration policies
- Transfer procedures
- DNSSEC requirements
- Dispute resolution procedures
- Registrar agreement terms
