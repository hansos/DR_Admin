# Hosting Plans

Configure hosting packages offered to customers with resource limits and features.

## How to Access

Navigate to **Infrastructure > Hosting Plans** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Plan Name | Name of the hosting plan. |
| Disk Space | Storage allocation. |
| Bandwidth | Monthly transfer limit. |
| Domains | Maximum number of domains. |
| Email Accounts | Maximum email accounts. |
| Price | Monthly or annual price. |
| Actions | Edit, Duplicate, Delete. |

## Creating a Plan

1. Click **Add Plan**.
2. Enter the plan name and configure resource limits.
3. Set the price for each billing cycle.
4. Optionally link to a [Hosting Provider](hosting-providers.md) and [Hosting Panel](hosting-panels.md).
5. Click **Save**.
