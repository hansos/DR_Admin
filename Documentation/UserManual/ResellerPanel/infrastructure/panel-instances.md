# Panel Instances

Manage individual control panel installations deployed on your servers.

## How to Access

Navigate to **Infrastructure > Panel Instances** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Instance Name | Friendly name for this installation. |
| Panel Type | The [Hosting Panel](hosting-panels.md) type. |
| Server | The [Server](servers.md) this panel is installed on. |
| URL | Admin panel URL. |
| Status | Online, Offline, Error. |
| Actions | Edit, Open Panel, Test, Delete. |

## Adding a Panel Instance

1. Click **Add Instance**.
2. Select the panel type and server.
3. Enter the admin URL and credentials.
4. Click **Save & Test** to verify connectivity.
