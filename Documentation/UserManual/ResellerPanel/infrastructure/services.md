# Services

Define and manage service types running on your infrastructure.

## How to Access

Navigate to **Infrastructure > Services** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Service Name | Name of the service (e.g., Apache, Nginx, Postfix, BIND). |
| Category | Web, Mail, DNS, Database, Other. |
| Servers | Number of servers running this service. |
| Actions | Edit, Delete. |

## Adding a Service

1. Click **Add Service**.
2. Enter the service name and select a category.
3. Click **Save**.
