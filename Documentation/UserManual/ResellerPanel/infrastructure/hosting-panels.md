# Hosting Panels

Define supported control panel types (e.g., cPanel, Plesk, DirectAdmin).

## How to Access

Navigate to **Infrastructure > Hosting Panels** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Panel Name | Name of the control panel type. |
| Vendor | Software vendor. |
| Instances | Number of [Panel Instances](panel-instances.md) deployed. |
| Actions | Edit, Delete. |

## Adding a Panel Type

1. Click **Add Panel Type**.
2. Enter the panel name and vendor.
3. Click **Save**.
