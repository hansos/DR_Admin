# Operating Systems

Manage the list of supported operating systems available for server provisioning.

## How to Access

Navigate to **Infrastructure > Operating Systems** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| OS Name | Operating system name (e.g., Ubuntu 24.04, Windows Server 2025). |
| Family | Linux, Windows, or Other. |
| Servers | Number of servers running this OS. |
| Actions | Edit, Delete. |

## Adding an Operating System

1. Click **Add OS**.
2. Enter the name and select the family.
3. Click **Save**.
