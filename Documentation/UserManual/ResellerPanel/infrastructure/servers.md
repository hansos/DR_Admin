# Servers

Manage your server inventory, view status, and assign services.

## How to Access

Navigate to **Infrastructure > Servers** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Server Name | Hostname or friendly name. |
| Server Type | Classification from [Server Types](server-types.md). |
| OS | Operating system from [Operating Systems](operating-systems.md). |
| IP Address | Primary IP address. |
| Provider | [Hosting Provider](hosting-providers.md) if applicable. |
| Status | Online, Offline, Maintenance. |
| Actions | View, Edit, Delete. |

## Adding a Server

1. Click **Add Server**.
2. Enter server name and select the type and OS.
3. Assign an IP address from [IP Addresses](ip-addresses.md) or enter a new one.
4. Optionally link to a hosting provider.
5. Click **Save**.
