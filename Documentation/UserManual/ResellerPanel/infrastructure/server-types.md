# Server Types

Define server type classifications (e.g., web server, mail server, DNS server).

## How to Access

Navigate to **Infrastructure > Server Types** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Type Name | Name of the server type. |
| Description | Brief description of this type's purpose. |
| Servers | Number of servers classified as this type. |
| Actions | Edit, Delete. |

## Adding a Server Type

1. Click **Add Type**.
2. Enter a name and description.
3. Click **Save**.
