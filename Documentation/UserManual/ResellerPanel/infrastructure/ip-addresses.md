# IP Addresses

Manage and allocate IP addresses across servers and services.

## How to Access

Navigate to **Infrastructure > IP Addresses** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| IP Address | The IPv4 or IPv6 address. |
| Type | IPv4 or IPv6. |
| Server | Server this IP is assigned to. |
| Purpose | Web, Mail, DNS, or General. |
| Status | Available, Assigned, Reserved. |
| Actions | Edit, Assign, Release, Delete. |

## Adding an IP Address

1. Click **Add IP**.
2. Enter the IP address and select the type.
3. Optionally assign to a server and set a purpose.
4. Click **Save**.
