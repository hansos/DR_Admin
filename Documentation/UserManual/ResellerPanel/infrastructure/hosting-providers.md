# Hosting Providers

Manage hosting provider accounts and API credentials.

## How to Access

Navigate to **Infrastructure > Hosting Providers** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Provider Name | Name of the hosting provider. |
| API Endpoint | Provider's API URL. |
| Servers | Number of servers managed through this provider. |
| Status | Connected, Disconnected, Error. |
| Actions | Edit, Test, Delete. |

## Adding a Provider

1. Click **Add Provider**.
2. Enter the provider name and API endpoint.
3. Enter API credentials.
4. Click **Save & Test** to verify connectivity.
