# Notifications

View and configure system notifications and alert preferences.

## How to Access

Navigate to **Alerts & Automation > Notifications** from the side menu.

## Page Layout

### Notification Inbox

A list of received notifications:

| Column | Description |
|--------|-------------|
| Date | When the notification was sent. |
| Type | Domain, Billing, Transfer, System. |
| Message | Notification summary. |
| Read | Whether it has been read. |
| Actions | Mark Read/Unread, Delete. |

### Notification Preferences

Configure which notifications you want to receive and how:

| Setting | Options |
|---------|---------|
| Domain Expiry | Email, In-App, Both, None. |
| Transfer Updates | Email, In-App, Both, None. |
| Payment Received | Email, In-App, Both, None. |
| Invoice Overdue | Email, In-App, Both, None. |
| System Alerts | Email, In-App, Both, None. |
