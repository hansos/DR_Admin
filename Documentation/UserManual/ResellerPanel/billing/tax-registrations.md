# Tax Registrations

Manage your company's tax registration numbers (e.g., VAT ID) per jurisdiction.

## How to Access

Navigate to **Billing & Finance > Tax Registrations** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Jurisdiction | The tax jurisdiction this registration applies to. |
| Tax ID | Your company's tax registration number (e.g., VAT number). |
| Valid From | Start date of the registration. |
| Valid To | Expiration date (if applicable). |
| Status | Active or Expired. |
| Actions | Edit, Delete. |

## Adding a Registration

1. Click **Add Registration**.
2. Select the [Tax Jurisdiction](tax-jurisdictions.md).
3. Enter your tax registration number.
4. Set validity dates.
5. Click **Save**.

## Important Notes

- Tax registration numbers appear on invoices for the relevant jurisdictions.
- Ensure registrations are kept up to date for tax compliance.
