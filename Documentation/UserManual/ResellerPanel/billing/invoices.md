# Invoices

View, create, and manage customer invoices for domain services.

## How to Access

Navigate to **Billing & Finance > Invoices** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Invoice # | Unique invoice identifier. |
| Customer | Customer the invoice is billed to. |
| Issue Date | Date the invoice was created. |
| Due Date | Payment deadline. |
| Amount | Total invoiced amount. |
| Paid | Amount paid so far. |
| Status | Draft, Sent, Paid, Partially Paid, Overdue, Cancelled. |
| Actions | View, Edit, Send, Record Payment, Download PDF. |

## Creating an Invoice

1. Click **New Invoice**.
2. Select a customer.
3. Add line items (domains, services, or custom items).
4. The system calculates taxes automatically based on [Tax Rules](tax-rules.md).
5. Click **Save as Draft** or **Save & Send**.
