# Orders

Track and manage domain registration, renewal, and transfer orders.

## How to Access

Navigate to **Billing & Finance > Orders** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Order # | Unique order identifier. |
| Customer | Customer who placed the order. |
| Type | Registration, Renewal, Transfer, or Service. |
| Items | Summary of items in the order. |
| Total | Order total including taxes. |
| Status | Pending, Processing, Completed, Failed, Cancelled. |
| Date | Date the order was created. |
| Actions | View, Process, Cancel. |

## Creating an Order

Orders are typically created from an accepted [Quote](../dashboard/new-quote.md) or directly:
1. Click **New Order**.
2. Select customer and add items.
3. Review pricing and taxes.
4. Submit the order.
