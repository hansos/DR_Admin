# Currencies

Manage supported currencies and exchange rates for multi-currency billing.

## How to Access

Navigate to **Billing & Finance > Currencies** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Currency Code | ISO 4217 code (e.g., USD, EUR, NOK). |
| Name | Full currency name. |
| Symbol | Display symbol (e.g., $, kr). |
| Exchange Rate | Rate relative to the base currency. |
| Default | Whether this is the system's base currency. |
| Actions | Edit, Delete. |

## Adding a Currency

1. Click **Add Currency**.
2. Enter the currency code, name, and symbol.
3. Set the exchange rate relative to your base currency.
4. Click **Save**.
