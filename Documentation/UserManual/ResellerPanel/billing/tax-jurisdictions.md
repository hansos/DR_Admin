# Tax Jurisdictions

Define geographic tax regions and their applicable tax rates.

## How to Access

Navigate to **Billing & Finance > Tax Jurisdictions** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Jurisdiction Name | Name of the tax region (e.g., Norway, EU, California). |
| Country | Associated country. |
| Region | State, province, or sub-region (if applicable). |
| Standard Rate | Default tax rate percentage. |
| Actions | View, Edit, Delete. |

## Adding a Jurisdiction

1. Click **Add Jurisdiction**.
2. Enter the jurisdiction name, country, and optional region.
3. Set the standard tax rate.
4. Click **Save**.
