# Coupons

Create and manage discount coupons and promotional codes.

## How to Access

Navigate to **Billing & Finance > Coupons** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Code | The coupon code customers enter at checkout. |
| Discount | Percentage or fixed amount discount. |
| Valid From | Start date of the coupon's validity. |
| Valid To | Expiration date. |
| Usage Limit | Maximum number of times the coupon can be used. |
| Used | Number of times the coupon has been redeemed. |
| Status | Active, Expired, or Disabled. |
| Actions | Edit, Disable, Delete. |

## Creating a Coupon

1. Click **Add Coupon**.
2. Enter a unique coupon code.
3. Choose discount type (percentage or fixed amount) and value.
4. Set validity dates and usage limit.
5. Optionally restrict to specific TLDs, customers, or products.
6. Click **Save**.
