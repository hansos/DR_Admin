# Order Tax Snapshots

View the tax calculations captured at the time of order placement for audit purposes.

## How to Access

Navigate to **Billing & Finance > Order Tax Snapshots** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Order # | The order this snapshot belongs to. |
| Customer | Customer who placed the order. |
| Jurisdiction | Tax jurisdiction applied. |
| Tax Rate | Rate applied at the time. |
| Tax Amount | Calculated tax amount. |
| Rule Used | The [Tax Rule](tax-rules.md) that was matched. |
| Snapshot Date | When the tax was calculated. |
| Actions | View Details. |

## Why Snapshots Exist

Tax rates and rules can change over time. Snapshots preserve the exact tax calculation at the time of the order, ensuring:

- **Compliance** â€” Invoices reflect the rate that was valid when the order was placed.
