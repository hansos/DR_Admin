# Tax Categories

Classify products and services into tax categories for correct tax treatment.

## How to Access

Navigate to **Billing & Finance > Tax Categories** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Category Name | Name of the tax category (e.g., Domain Registration, Hosting, Consulting). |
| Description | Brief description of what this category covers. |
| Products | Number of products assigned to this category. |
| Actions | Edit, Delete. |

## Adding a Category

1. Click **Add Category**.
2. Enter a category name and description.
3. Click **Save**.

## How Categories Are Used

Tax categories are referenced by [Tax Rules](tax-rules.md) to determine the applicable tax rate for a product in a given jurisdiction.
