# Billing Cycles

Configure recurring billing periods for domain services and subscriptions.

## How to Access

Navigate to **Billing & Finance > Billing Cycles** from the side menu.

## Page Layout

| Column | Description |
|--------|-------------|
| Cycle Name | Descriptive name (e.g., Annual, Biennial). |
| Period | Duration in months. |
| Auto-Invoice | Whether invoices are generated automatically before the cycle renews. |
| Advance Days | How many days before expiry the invoice is created. |
| Actions | Edit, Delete. |

## Creating a Billing Cycle

1. Click **Add Billing Cycle**.
2. Enter a name and period (in months).
3. Enable **Auto-Invoice** if invoices should be generated automatically.
4. Set the **Advance Days** for early invoice generation.
5. Click **Save**.
