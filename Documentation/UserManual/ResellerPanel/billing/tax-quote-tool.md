# Tax Quote Tool

Preview tax calculations for a potential order before finalizing.

## How to Access

Navigate to **Billing & Finance > Tax Quote Tool** from the side menu.

## Step-by-Step

1. Select a **Customer** (or enter a country/jurisdiction manually).
2. Add one or more **Products** with quantities.
3. Click **Calculate**.
4. Review the breakdown:
   - Subtotal per item.
   - Applicable tax jurisdiction and rate.
   - Tax amount per item and total.
   - Grand total.
