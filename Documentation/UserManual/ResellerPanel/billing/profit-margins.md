# Profit Margins

Analyze profit margins across products, TLDs, and customers.

## How to Access

Navigate to **Billing & Finance > Profit Margins** from the side menu.

## Page Layout

A report view showing margin analysis with the following dimensions:

| Dimension | Description |
|-----------|-------------|
| By TLD | Margin per top-level domain. |
| By Product | Margin per product type (registration, renewal, transfer). |
| By Customer | Margin per customer account. |

### Columns

| Column | Description |
|--------|-------------|
| Item | TLD, product, or customer name. |
| Revenue | Total sell-price revenue. |
| Cost | Total cost price. |
| Margin | Revenue minus cost. |
| Margin % | Margin as a percentage of revenue. |
