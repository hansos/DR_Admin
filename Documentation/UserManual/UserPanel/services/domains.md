# Domains

Review your registered domains and access lifecycle tools.

## How to Access

Navigate to **My Services > Domains** from the side menu.

## Page Layout

A paginated, filterable list of all domains registered under your account.

### Filters

| Filter | Description |
|--------|-------------|
| Search | Free-text search by domain name. |
| Status | Filter by domain status (Active, Expired, Pending, etc.). |
| Page size | Number of domains shown per page. |

### Table Columns

| Column | Description |
|--------|-------------|
| Domain Name | Fully qualified domain name. |
| Status | Current registration status. |
| Expiry Date | When the domain registration expires. |
| Auto-Renew | Whether automatic renewal is enabled. |
| Actions | View details, manage DNS, renew. |

## Available Actions

- **View Details** — Open the domain detail page to manage contacts, nameservers, DNS, and transfer lock.
- **Manage DNS** — Jump to DNS record management for the domain.
- **Renew** — Initiate a renewal for the domain.
