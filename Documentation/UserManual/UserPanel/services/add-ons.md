# Add-ons / SSL / Services

Overview of optional services, SSL certificates, and add-ons connected to your subscriptions.

## How to Access

Navigate to **My Services > Add-ons / SSL / Services** from the side menu.

## Page Layout

A paginated, filterable list of all optional service subscriptions.

### Filters

| Filter | Description |
|--------|-------------|
| Status | Filter by subscription status (Active, Suspended, Pending, etc.). |
| Billing cycle | Filter by billing cycle period. |
| Page size | Number of items shown per page. |

### Table Columns

| Column | Description |
|--------|-------------|
| Service Name | Name of the add-on or service. |
| Type | SSL, DNS add-on, email service, etc. |
| Status | Current subscription status. |
| Billing Cycle | Renewal period. |
| Next Billing | Date of the next billing event. |
| Actions | View details. |
