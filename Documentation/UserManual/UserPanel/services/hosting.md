# Hosting

Review your hosting subscriptions, status, and plan information.

## How to Access

Navigate to **My Services > Hosting** from the side menu.

## Page Layout

A paginated, filterable list of all hosting subscriptions under your account.

### Filters

| Filter | Description |
|--------|-------------|
| Status | Filter by subscription status (Active, Suspended, Pending, etc.). |
| Billing cycle | Filter by billing cycle period. |
| Page size | Number of subscriptions shown per page. |

### Table Columns

| Column | Description |
|--------|-------------|
| Hosting Plan | Name of the hosting package. |
| Status | Current subscription status. |
| Billing Cycle | Renewal period (monthly, annual, etc.). |
| Next Billing | Date of the next billing event. |
| Actions | View details. |

## Available Actions

- **View Details** — Open the hosting detail page to see server info, resource usage, and control panel access.
