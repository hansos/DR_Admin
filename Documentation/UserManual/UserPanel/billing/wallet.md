# Wallet

Track your credit balance and view transaction history.

## How to Access

Navigate to **Billing > Wallet** from the side menu.

## Page Layout

### Current Balance

A card displaying your current wallet credit balance.

### Transactions

A table listing all wallet transactions:

| Column | Description |
|--------|-------------|
| Date | Date of the transaction. |
| Type | Credit, Debit, Refund, etc. |
| Amount | Transaction amount. |
| Description | Brief description of the transaction. |
| Balance | Wallet balance after the transaction. |

## How Credit Works

- Credit can be applied to future invoices and orders.
- Refunds may be deposited into your wallet balance.
- When paying for an order, wallet credit is applied before charging your payment method.
