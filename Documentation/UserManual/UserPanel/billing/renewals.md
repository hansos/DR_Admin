# Renewals

Manage auto-renew settings and upcoming subscription renewals.

## How to Access

Navigate to **Billing > Renewals** from the side menu.

## Page Layout

A table listing all active subscriptions (domains, hosting, add-ons) and their renewal status.

### Table Columns

| Column | Description |
|--------|-------------|
| ID | Subscription identifier. |
| Status | Current subscription status. |
| Next Billing | Date of the next billing event. |
| Amount | Renewal amount. |
| Auto Renew | Whether automatic renewal is enabled (toggle). |
| Actions | Renew now, toggle auto-renew. |

### Actions

- **Toggle Auto-Renew** — Enable or disable automatic renewal for a subscription.
- **Renew Now** — Manually trigger an immediate renewal and generate an invoice.

## Important Notes

- Subscriptions with auto-renew enabled are renewed automatically before the expiry date.
- An invoice is generated in advance per the configured billing cycle.
