# Payment Methods

Manage saved payment instruments. The system automatically selects the configured default gateway.

## How to Access

Navigate to **Billing > Payment Methods** from the side menu.

## Page Layout

### Saved Instruments

A table listing your saved payment instruments:

| Column | Description |
|--------|-------------|
| Payment Instrument | The payment method name and type (e.g., Credit Card, PayPal). |
| State | Active or Disabled. |
| Created | Date the instrument was added. |
| Actions | Edit, Remove. |

### Adding a Payment Instrument

1. Click **Add payment instrument**.
2. Select the instrument type from the available options defined by your service provider.
3. Enter the required details.
4. Click **Save**.

## How It Works

When you make a payment, you select a saved payment instrument. The system automatically routes the transaction through the default payment gateway configured for that instrument — you do not need to choose a gateway.
