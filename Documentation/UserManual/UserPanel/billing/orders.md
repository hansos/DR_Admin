# Orders

Review your orders and line details.

## How to Access

Navigate to **Billing > Orders** from the side menu.

## Page Layout

A paginated list of all orders placed on your account.

### Order Summary

Each order entry displays:

| Field | Description |
|-------|-------------|
| Order # | Unique order identifier. |
| Date | Date the order was placed. |
| Items | Summary of items in the order. |
| Total | Order total including taxes. |
| Status | Pending, Processing, Completed, Failed, or Cancelled. |

### Actions

- **View** — Open the order to see full line item details, pricing, and associated invoice.

## Pagination

Use the **Previous** and **Next** buttons to navigate between pages of orders.
