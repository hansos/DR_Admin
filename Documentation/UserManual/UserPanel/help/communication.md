# Communication

Use the communication center to read support communication and send replies.

## Where to find it

`Help -> Communication`

## What you can do

- View your communication threads.
- Open a thread and read the full conversation timeline.
- Mark inbound messages as read.
- Send a reply from inside the thread.

## Security scope

The communication API is customer-scoped. You only have access to your own customer/user threads.
