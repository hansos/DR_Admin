# Support

Track support requests and create new tickets.

## How to Access

Navigate to **Help > Support** from the side menu.

## Page Layout

### Create a Ticket

A form at the top of the page to submit a new support request:

| Field | Description |
|-------|-------------|
| Subject | Brief description of the issue (max 150 characters). |
| Priority | Low, Normal, or High. |
| Message | Detailed description of your request. |

Click **Submit** to create the ticket.

### Ticket List

A list of your existing support tickets:

| Column | Description |
|--------|-------------|
| Ticket # | Unique ticket identifier. |
| Subject | Short description of the issue. |
| Priority | Low, Normal, or High. |
| Status | Open, In Progress, Awaiting Reply, Resolved, Closed. |
| Created | Date the ticket was opened. |
| Last Updated | Date of the most recent activity. |
| Actions | View, Reply. |

### Actions

- **View** — Open the ticket thread to see the full conversation.
- **Reply** — Add a follow-up message to an existing ticket.
- **Refresh** — Reload the ticket list.
