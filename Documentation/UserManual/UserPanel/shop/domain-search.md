# Domain Search

Start your purchase by searching for available domain names, then bundle hosting and services.

## How to Access

Navigate to **Shop > Domain Search** from the side menu.

## Page Layout

### Search

Enter one or more domain names to check availability across supported TLDs. Results display availability status and pricing for each name.

### Order Builder

After finding an available domain, you can:

1. **Add to order** — Add the domain to your current order.
2. **Bundle hosting** — Attach a hosting plan to the domain.
3. **Bundle services** — Add SSL certificates or other add-ons.

### Current Order

A summary of items added to the current order is displayed, showing domain names, bundled services, prices, and totals.

| Action | Description |
|--------|-------------|
| Remove item | Remove a domain or service from the order. |
| Proceed to Checkout | Continue to the [Checkout](checkout.md) page. |
| Delete order | Discard the entire order (only if unpaid). |
