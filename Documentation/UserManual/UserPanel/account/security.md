# Security

Review successful and failed sign-in attempts, sessions, and device activity.

## How to Access

Navigate to **Account > Security** from the side menu.

## Page Layout

A table showing your recent sign-in history:

| Column | Description |
|--------|-------------|
| Time | Date and time of the sign-in attempt. |
| Result | Success or Failed. |
| IP | IP address of the client. |
| Device | Browser and device information. |

### Actions

- **Refresh** — Reload the session list to see the latest activity.

## Security Tips

- Review this page regularly to detect unauthorized access attempts.
- If you see sign-in attempts from unfamiliar locations or devices, change your password immediately and enable [Two-Factor Authentication](two-factor-auth.md).
