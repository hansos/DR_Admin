# Notifications

Configure renewal reminders, payment reminders, and the in-app alert center.

## How to Access

Navigate to **Account > Notifications** from the side menu.

## Page Layout

### Notification Preferences

A set of toggle switches to control which notifications you receive:

| Setting | Description |
|---------|-------------|
| Renewal reminders | Receive reminders before domains or services are due for renewal. |
| Payment reminders | Receive reminders for unpaid invoices. |

### In-App Alert Center

A feed of recent in-app notifications and alerts. Mark items as read or dismiss them.

## Saving Changes

Toggle the desired notification preferences and click **Save** to apply. Changes take effect immediately.
