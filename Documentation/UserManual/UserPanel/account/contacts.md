# Contacts

Manage billing, technical, and administrative contact persons.

## How to Access

Navigate to **Account > Contacts** from the side menu.

## Page Layout

A list of all contact persons associated with your account. Each contact shows name, email, phone, position, and role flags.

### Actions

| Button | Description |
|--------|-------------|
| **Add contact** | Open the dialog to create a new contact person. |
| **Edit** | Modify an existing contact's details. |
| **Delete** | Remove a contact person. |

### Contact Person Fields

| Field | Description |
|-------|-------------|
| First Name * | Contact's first name (required). |
| Last Name * | Contact's last name (required). |
| Email * | Contact email address (required). |
| Phone * | Contact phone number (required). |
| Position | Job title or role. |
| Department | Organizational department. |
| Notes | Additional notes. |

### Role Toggles

| Toggle | Description |
|--------|-------------|
| Primary | Mark as the primary contact for your account. |
| Active | Enable or disable the contact. |
| Default Owner | Use as the default domain registrant contact. |
| Default Billing | Use as the default billing contact for domains. |
| Default Tech | Use as the default technical contact for domains. |
| Default Admin | Use as the default administrative contact for domains. |
| Domain Global | Apply as a global default across all your domains. |
