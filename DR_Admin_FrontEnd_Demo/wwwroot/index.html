Welcome to DR Admin Demo

A production-like demo frontend for domain registration and hosting management.

This demo showcases how a real-world reseller or end-user frontend could be structured for managing domains and hosting services.

This is a demonstration implementation - not a production product. It shows typical layout, navigation patterns, and user workflows.

Development Note

This frontend uses simulated API responses. To connect to the real DR_Admin backend:

  • Update the API endpoints in Controllers/*Controller.cs
  • Configure the backend URL in appsettings.json
  • Implement actual HTTP calls in the TypeScript api-client.ts module
  • Add authentication tokens and proper session management
